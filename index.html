MWEB Roadmap

MWEB

  • MWEB is set to be the largest upgrade to the Litecoin network ever, promising to provide fungibility for all, while improving LTC's scalability. Due to the size of the change, it has been in progress for 2 years now, but the end is in sight! This timeline provides our best estimates for when the final steps will take place, but please keep in mind that everything is subject to change.
  • Mar 15 2021

    Code Complete

    libmw is a completely self-contained project with clearly defined set of APIs that all calls from the existing Litecoin project go through. It currently uses cmake and builds as a shared library, but to maintain consistency with the existing build & release process, we will eventually turn this into a makefile project instead, and build it as a static library. I will create some docs later this week that will offer suggestions for how to go about reviewing the code, and will include API usage examples that you can use as a guide to understanding how libmw is used.
  • Q2 2021

    Reviews and Audits

    The code will be reviewed by LTC & BTC developers, mimblewimble experts, and professional security auditors. We will also be testing in parallel and fixing any issues found.
  • August 2021

    Release

    The first LTC release containing MWEB code. This will have the complete MWEB code, ready to be used by miners and node operators everywhere as soon as MWEB activates.
  • Q3 2021

    Miner signaling

    Once miners take the released code, they may begin signaling for MWEB activation right away. As soon as the activation threshold is met, the MWEB activation date will be locked-in.
  • Q4 2021

    MWEB activates!

    Enjoy using one of the most scalable, fungible cryptocurrencies ever. May God have mercy on the fiat currencies of the world.
